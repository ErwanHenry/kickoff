## Key Achievements

### Database Query
- **getMatchTeams function** (`src/lib/db/queries/players.ts`)
  - Queries match_players filtered by matchId and status='confirmed'
  - LEFT JOINs with users table for player names (COALESCE with guestName)
  - LEFT JOINs with player_stats for ratings (prefers group-specific stats)
  - Filters by team column ('A' or 'B')
  - Returns BalanceResult with teamA, teamB, diff, and algorithm
  - Calculates total scores per team
  - Defaults missing stats to 3.0, converts Decimal to number

### Teams Page Integration
- **Updated teams page** (`src/app/match/[id]/teams/page.tsx`)
  - Fetches actual team data from database when match is locked
  - Displays match header with title, location, date using FootballIcon
  - Applies design system colors (bg-chalk, text-pitch)
  - Sets isLocked={false} to allow drag-and-drop
  - Adds "Remélanger" button to regenerate teams
  - Formats date in French with date-fns
