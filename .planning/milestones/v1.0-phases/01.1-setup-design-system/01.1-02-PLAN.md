## Why Split?

The original plan had 8 tasks spanning multiple concerns:
- Wave 1: Colors audit
- Wave 2: Icon replacement
- Wave 3: Typography
- Wave 4: Tokens & badges
- Waves 5-8: Retrofits

**Problems:**
- Too many tasks for a single plan (recommended: 3-5 tasks)
- Mixed concerns made verification difficult
- Hard to track progress across different areas

**Benefits of splitting:**
- Each plan focuses on one concern (colors, icons/typography, tokens)
- Easier to verify completion
- Better isolation if issues arise
- Clearer success criteria per plan
