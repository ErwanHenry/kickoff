  <action>
Extend src/lib/db/queries/groups.ts with two new query functions:

1. **getGroupLeaderboard(groupId: string)** - Get leaderboard ranked by avg_overall
   - Join: player_stats → users
   - Filter: player_stats.groupId equals groupId
   - Select: userId as id, userName, matchesPlayed, attendanceRate, avgOverall, totalRatingsReceived
   - Order by: avgOverall DESC NULLS LAST
   - Limit: 50 (reasonable max for group size)
   - Convert Decimal to number for avgOverall and attendanceRate
   - Return: Array of { id, name, matchesPlayed, attendanceRate, avgOverall, totalRatingsReceived }
   - Handle case: No stats yet → Return empty array (players with 0 ratings not shown until rated)

2. **getGroupMatchHistory(groupId: string, limitCount: number = 20)** - Get group's match history
   - Join: matches → groups
   - Filter: matches.groupId equals groupId AND matches.status IN ('played', 'rated')
   - Select: id, title, date, location, status, shareToken, scoreTeamA, scoreTeamB
   - Order by: date DESC
   - Limit: limitCount (default 20)
   - Return: Array of { id, title, date, location, status, shareToken, scoreTeamA, scoreTeamB }

3. **getGroupMembers(groupId: string)** - Get all group members with roles
   - Join: group_members → users
   - Filter: group_members.groupId equals groupId
   - Select: userId as id, userName, role, joinedAt
   - Order by: joinedAt ASC (first members first)
   - Return: Array of { id, name, role, joinedAt }

Follow the Drizzle query builder pattern from players.ts. Use sql operators for type conversions.
