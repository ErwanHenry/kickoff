  <action>
    Create src/lib/stats.ts with utility functions:

    calculateIncrementalAverage(
      oldAverage: number,
      oldCount: number,
      newValue: number
    ): number
    - Formula: ((oldAverage * oldCount) + newValue) / (oldCount + 1)
    - Round to 2 decimal places
    - Return result
    - Example: ((3.5 * 10) + 4) / 11 = 3.59

    calculateWeightedOverall(
      technique: number,
      physique: number,
      collectif: number
    ): number
    - Formula: (technique * 0.4) + (physique * 0.3) + (collectif * 0.3)
    - Round to 2 decimal places
    - Return result
    - Example: (4.0 * 0.4) + (3.5 * 0.3) + (3.8 * 0.3) = 3.79

    updatePlayerStatsFromRatings(
      ratings: Array<{ technique, physique, collectif }>,
      currentStats: {
        avgTechnique: string,
        avgPhysique: string,
        avgCollectif: string,
        totalRatingsReceived: number
      }
    ): {
      avgTechnique: string,
      avgPhysique: string,
      avgCollectif: string,
      avgOverall: string,
      totalRatingsReceived: number
    }
    - For each axis, calculate incremental average using all new ratings
    - Average new ratings per axis, then apply incremental formula
    - Calculate weighted overall from new three-axis averages
    - Increment totalRatingsReceived by ratings.length
    - Return formatted as strings (2 decimal places)

    Helper: parseDecimal(value: string): number
    - Parse Drizzle decimal string to number
    - Handle null/undefined → return 0
  </action>

    getPlayerStats(userId: string, groupId?: string):
    - Query player_stats table where userId = userId
    - If groupId provided: filter by groupId also
    - Use COALESCE for groupId filter (null matches global stats)
    - Return single record or null
    - Fields: all player_stats columns

    createOrUpdatePlayerStats(
      userId: string,
      groupId: string | null,
      ratings: Array<{ technique, physique, collectif }>
    ): Promise player stats record
    - Check if stats record exists for (userId, groupId)
    - If exists: fetch current stats, call updatePlayerStatsFromRatings, update record
    - If not exists: create new record with initial values from ratings
    - For creation: avg = average of ratings per axis, total = ratings.length
    - Return updated/created record
    - Use Drizzle: db.insert().onConflictDoUpdate() for upsert

    updateMatchRatedStatus(matchId: string, db: DrizzleDB):
    - Count distinct rater_ids in ratings table for this match
    - Count confirmed players in match_players for this match
    - If (rater_count / confirmed_count) >= 0.5:
      - Update matches table: set status = "rated"
    - Return true if updated to "rated", false otherwise
    - Use transaction caller's db instance

    getMatchRatersCount(matchId: string):
    - Count distinct rater_id from ratings where matchId
    - Return count
    - Used for progress indicator

    getMatchConfirmedCount(matchId: string):
    - Count match_players where matchId AND status = "confirmed" AND attended = true
    - Return count
    - Used for 50% threshold calculation
  </action>
